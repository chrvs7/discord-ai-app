# discord-ai-bot

## Overview

A self-hosted Discord Profile App bot that uses Google Gemini 2.0 Flash (free tier) to answer prompts via `/ai <prompt>`. The bot has a witty, sarcastic personality — blunt and mildly condescending, but always actually useful. It maintains per-session conversation history so follow-up questions work naturally. Costs nothing to run.

## Prerequisites

- Python 3.11+
- pip
- Docker (optional, for containerized deployment)
- A Discord account with access to the Developer Portal
- A Google account with access to Google AI Studio

## Quickstart (Local)

1. Clone the repo and `cd` into it:
   ```
   git clone <repo-url>
   cd discord-ai-bot
   ```
2. Copy `.env.example` to `.env` and fill in both keys:
   ```
   cp .env.example .env
   ```
3. Create and activate a virtual environment:
   ```
   python -m venv .venv
   source .venv/bin/activate   # Windows: .venv\Scripts\activate
   ```
4. Install dependencies:
   ```
   pip install -r requirements.txt
   ```
5. Start the bot:
   ```
   python bot.py
   ```

## Discord Setup

1. Go to [discord.com/developers/applications](https://discord.com/developers/applications) and create a **New Application**.
2. Navigate to the **Bot** tab, click **Add Bot**, and enable **Message Content Intent**.
3. Copy the bot token into `.env` as `DISCORD_TOKEN`.
4. Go to **OAuth2 > URL Generator**, select scopes: `bot` and `applications.commands`.
5. Select permissions: **Send Messages**, **Embed Links**, **Attach Files**, **Use Slash Commands**.
6. Copy the generated URL and use it to invite the bot to your server.

## Gemini Setup

1. Go to [aistudio.google.com](https://aistudio.google.com).
2. Click **Get API Key > Create API Key**.
3. Copy the key into `.env` as `GEMINI_API_KEY`.
4. Free tier limits: 1,500 requests/day, 15 requests/minute.

## Docker Deployment

Build the image:
```
docker build -t discord-ai-bot .
```

Run as a background container that restarts automatically:
```
docker run -d \
  --name discord-ai-bot \
  --restart unless-stopped \
  --env-file .env \
  discord-ai-bot
```

Tail logs:
```
docker logs -f discord-ai-bot
```

Stop or restart:
```
docker stop discord-ai-bot
docker restart discord-ai-bot
```

Using docker-compose (starts in background):
```
docker compose up -d
```

Rebuild and restart after code changes:
```
docker compose up -d --build
```

Tail compose logs:
```
docker compose logs -f
```

## Commands

| Command          | Description                                        | Example                             |
|------------------|----------------------------------------------------|-------------------------------------|
| `/ai <prompt>`   | Send a prompt to the AI and get a reply            | `/ai explain recursion like I'm 5`  |
| `/ai-clear`      | Wipe your conversation history and start fresh     | `/ai-clear`                         |

## Profile App Setup

The bot can be installed as a Discord Profile App — accessible from any server or DM without a traditional bot invite — by enabling **User Install** under **OAuth2 > Authorization Methods** in the Developer Portal.
